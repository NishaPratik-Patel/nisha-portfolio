Smart Library Navigation System

Developed an intelligent system guiding customers to bookshelves using a Pudu service robot, barcode scanning, and computer vision. Reduced book search time and improved user experience.

In large libraries, customers often struggled to find books due to many shelves and limited staff assistance. I created a navigation map using the Pudu robot, mapping the library and assigning precise shelf numbers so the robot stops exactly at each shelf. Built a webcam-based application using Python, OpenCV, and Flask. Customers search for a book, get a barcode receipt, and the robot navigates them to the correct location. Wire Damage Detection

Real-time computer vision system detecting wire defects with multiple camera feeds and deep learning.

In manufacturing, detecting wire defects during bobbin winding was challenging due to inconsistent visual inspection and variable lighting. Developed a system with Python, OpenCV, Flask, TensorFlow, and Keras. Streams four webcam feeds, performs real-time defect detection, and triggers alerts to stop the robot or notify the operator. Improved defect detection accuracy and reduced reliance on manual inspection.
