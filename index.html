AI & Software Projects

Smart Library Navigation System

In large libraries, customers often struggled to find books due to many shelves and limited staff assistance. I developed an intelligent system that guides customers directly to the correct bookshelf using automation and computer vision. First, I created a navigation map using the Pudu service robot, mapping the library and assigning precise shelf numbers so the robot could stop exactly at each shelf. Then, I built a webcam-based application using Python, OpenCV, and Flask. Customers search for a book on a PC, generating a barcode receipt. The webcam scans the barcode, retrieves the shelf number, and sends it to the Pudu robot via API, which navigates the customer to the correct location. The system significantly reduced book search time, improved user experience, and effectively integrated computer vision, robotics, and web technologies in a real-world environment.

Python OpenCV Flask Robotics API

Wire Damage Detection

In a manufacturing environment, detecting wire defects during the bobbin winding process was challenging due to inconsistent visual inspection and variable lighting conditions. I developed a real-time computer vision system using Python, OpenCV, and Flask to stream four webcam feeds on a single dashboard for multi-angle monitoring. Using TensorFlow and Keras, I trained an object detection model with datasets annotated in LabelImg to classify OK and defective wires. To handle varying lighting conditions, I applied data augmentation techniques including brightness, shadow, and contrast variations. The system performs automatic defect detection and triggers alerts to stop the robot or notify the operator when defects are found. This solution significantly improved defect detection accuracy, reduced reliance on manual inspection, and enhanced overall quality control in the manufacturing process.

Python OpenCV TensorFlow Keras Flask

AI Question Answering Engine

Users often spent too much time searching manually through documents and websites, and keyword-based search frequently returned irrelevant results. I developed an intelligent semantic search engine using a BERT NLP model trained on a custom dataset to understand the meaning behind user questions. Both frontend and backend were built using C# .NET. The backend converts user questions into embeddings, compares them with document embeddings, ranks results using similarity scores, and returns the top 5 most relevant answers with links to the source documents. This system significantly reduced search time, improved answer accuracy, and enhanced user experience by making information retrieval faster and more reliable.

BERT NLP Machine Learning C# .NET

AI Meeting Transcription System

In many meetings, it was difficult to track who said what, and manually creating transcripts from audio recordings was time-consuming. I developed an AI system that automatically converts recorded meeting audio into structured conversations. Using speaker diarization, the system separates up to three speakers in mono audio and two in stereo audio. After separation, speech recognition transcribes each speaker’s voice, which is then displayed in a chat-style interface, making discussions easy to follow. This solution automated meeting transcription, reduced manual note-taking, and improved clarity, enabling teams to quickly review conversations and understand each participant’s contributions.

Speech Recognition Speaker Diarization Python
